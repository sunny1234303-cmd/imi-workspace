# 키워드 분석 도구 - Google Apps Script

구글 시트에서 직접 네이버 API를 호출하여 키워드 분석을 수행하는 커스텀 메뉴입니다.

## 설치 방법

### 1. Apps Script 열기
1. 구글 시트 열기: [키워드 분석 시트](https://docs.google.com/spreadsheets/d/1rJ1PD_IHEQNtaZhedJvyilNgRFE36tOl8yN66xhHq8k)
2. 메뉴: **확장 프로그램** > **Apps Script**

### 2. 코드 붙여넣기
1. 기본 `Code.gs` 파일의 내용을 모두 삭제
2. `Code.gs` 파일의 전체 내용을 복사하여 붙여넣기
3. **Ctrl+S** (저장)

### 3. 권한 승인
1. 시트로 돌아가서 **새로고침** (F5)
2. 상단에 **🔍 키워드 분석** 메뉴가 나타남
3. 아무 메뉴 클릭 시 권한 승인 요청
4. **고급** > **프로젝트명(안전하지 않음)으로 이동** > **허용**

### 4. API 키 설정
1. **🔍 키워드 분석** > **⚙️ API 설정** 클릭
2. 네이버 API 키 입력:
   - 데이터랩: Client ID, Client Secret
   - 검색광고: Access License, Secret Key, Customer ID
3. **저장** 클릭

## 사용 방법

### 키워드 검색량 분석
```
🔍 키워드 분석 > 📊 키워드 검색량 분석
```
- 키워드 입력 (쉼표 구분, 최대 5개)
- 연관 키워드 포함 여부 선택
- 결과: `Raw_키워드` 탭에 저장

### 트렌드 분석
```
🔍 키워드 분석 > 📈 키워드 트렌드 분석
```
- 키워드 입력
- 분석 기간 선택 (7일 ~ 1년)
- 디바이스 선택 (전체/PC/모바일)
- 결과: `Raw_트렌드` 탭에 저장

### 대시보드 새로고침
```
🔍 키워드 분석 > 🔄 대시보드 새로고침
```
- 수식 강제 재계산

## 메뉴 구조

```
🔍 키워드 분석
├── 📊 키워드 검색량 분석  ← 네이버 검색광고 API
├── 📈 키워드 트렌드 분석  ← 네이버 데이터랩 API
├── ────────────────────
├── 🔄 대시보드 새로고침
├── ────────────────────
└── ⚙️ API 설정
```

## 주의사항

1. **API 키 보안**: 스크립트 속성에 저장되어 공유되지 않음
2. **할당량**: 네이버 API 일일 호출 제한 있음 (데이터랩 1,000회, 검색광고 API는 계정별 상이)
3. **처음 실행 시**: 권한 승인 필요

## 문제 해결

### "API 설정이 필요합니다" 오류
→ **⚙️ API 설정**에서 키 입력 확인

### 권한 오류
→ 시트 새로고침 후 재시도

### API 오류
→ API 키가 올바른지, 네이버 계정에서 API가 활성화되어 있는지 확인
