**[지원동기]**

올리브영이 단순한 유통 채널을 넘어 '뷰티 라이프스타일 플랫폼'으로 자리잡는 과정에서, 저는 브랜드가 어떻게 소비자의 일상 언어가 되는지를 직접 경험하고 싶었습니다.

YBM 퍼블위즈에서 내셔널지오그래픽 브랜드를 담당하며 한 가지를 깨달았습니다. 브랜드 인지도가 높아도, '지금 이 사람'에게 '지금 이 이유'로 닿지 않으면 전환은 일어나지 않는다는 것. 새학기 시즌 캠페인에서 타깃을 '독자'가 아닌 '초등 자녀를 둔 학부모'로 재정의하고, LTV 높은 신규 고객 유치에 집중한 결과 신규 구매 고객 수 YoY 47.4% 상승, 월 KPI 153%를 달성했습니다. 이 경험이 올리브영의 IMC 캠페인 매니저 역할과 정확히 맞닿아 있다고 생각합니다.

---

**[직무 역량]**

**① 전략 → 실행 → 인사이트로 이어지는 캠페인 루프**

저의 캠페인 방식은 세 단계로 구성됩니다. 먼저 소비자 데이터와 시즌 트렌드를 분석해 '왜 지금 이 메시지인가'를 정의합니다. 과월호 골라담기 캠페인에서는 검색 트래픽 하락이 단일 채널 의존 문제임을 파악하고, 카카오 채널(7,300명 DB)과 네이버 스마트스토어 CRM을 연동해 비수기 매출 203% 상승, 브랜드 검색 유입률 143% 상승을 만들었습니다.

실행 이후에는 반드시 'Retrace'합니다. 어린이날 콜라보 캠페인에서 신제품 전환율이 기존 제품 가격 유사성으로 저하된 문제를 데이터로 확인했고, 이 인사이트는 다음 캠페인의 가격 전략과 타깃 연령대 집중화로 연결됐습니다. 이 루프가 올리브영의 "결과를 다음 캠페인 전략 개선으로 연결"하는 직무 방향과 일치한다고 생각합니다.

**② 멀티채널 통합 실행력**

온라인 채널(네이버 검색광고, 카카오 채널, 인스타그램), 인플루언서(키즈 인플루언서 11명 직접 컨택·계약·콘텐츠 방향 관리), 파트너사 콜라보(그란데클립 사입 단가 38% 절감)까지 단일 캠페인 안에서 채널별 역할을 설계하고 운영한 경험이 있습니다. 각 채널을 독립적으로 운영하는 것이 아니라, 하나의 캠페인 목적 아래 채널 간 시너지를 설계하는 방식으로 접근해왔습니다.

**③ AI 기반 마케팅 실행 가속화**

트렌드 분석 3시간 → 1분, 월간 리포트 4시간 → 클릭 1번, 검색 키워드 분석 3시간 → 10분으로 단축하는 자동화 구조를 직접 설계했습니다. 단순한 업무 효율화가 아니라, 반복 분석에 쓰이던 시간을 '다음 캠페인 아이디어를 기획하는 시간'으로 전환하는 것이 목적이었습니다. 올리브영처럼 빠르게 변화하는 트렌드 환경에서, 이 역량은 실행 속도를 높이는 실질적인 경쟁력이 된다고 생각합니다.

---

**[포부]**

올리브영의 캠페인이 소비자에게 단순한 프로모션이 아닌 '브랜드 경험'으로 남을 수 있도록, 전략부터 현장 실행까지 전 과정을 책임지는 매니저가 되고 싶습니다. 뷰티 업계 인하우스 경험은 없지만, 브랜드가 소비자와 어떻게 관계를 맺는가에 대한 본질적인 관점과 실행 경험은 업종을 가리지 않습니다. 올리브영에서 그 관점을 뷰티 카테고리에 적용하고 싶습니다.

### 이력서 조정 포인트

**1. 커버/오프닝 문구 수정**
- 현재: "마케팅이 '작동'하는 구조를 만듭니다"
- 권장: 이 문구는 유지하되, 1페이지 Check Point에 아래 항목 추가 검토
  - ✔ IMC 캠페인 전 과정 기획·실행 (기획 100% / 제안 100% / 실행 100%)
  - ✔ 멀티채널 통합 운영 (검색·CRM·SNS·인플루언서)
  - (현재 4개 항목은 이미 잘 구성되어 있음)

**2. Project 순서 재정렬 (이 직무용)**
- IMC 캠페인 매니저 직무에는 아래 순서가 유리:
  - 01. 브랜드 캠페인 (새학기) → **브랜딩 전략 역량**
  - 02. 콜라보·인플루언서 + Retrace → **확산 전략 + 인사이트 루프**
  - 03. 검색광고·CRM 통합 → **퍼포먼스 관리 역량**
  - 04. AI 도입 + KPIs 대시보드 → **데이터 기반 운영**

**3. Project 03 (인플루언서) 결과 보강**
- 현재: "상위 노출" + "자산 확보" → 수치 없음
- 개선: 릴스 조회수, 저장수, 도달수 중 하나라도 추가 가능하면 보완
- 없다면 Retrace 섹션 (p.06)을 바로 연결해 "인사이트 도출 역량"으로 마무리하는 구조 유지

**4. Retrace 섹션 강조**
- 올리브영 직무 요건 "결과를 다음 캠페인 전략으로 연결"과 정확히 매칭됨
- 현재 p.06에 잘 구성되어 있으나, 앞 페이지(Project 03)와의 연결이 명시적이지 않음
- "이 캠페인에서 발견한 문제가 다음 전략을 바꿨다"는 흐름을 페이지 전환 시 한 줄로 표시 권장

**5. 뷰티 업계 경험 부재 대응**
