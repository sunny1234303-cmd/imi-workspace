# 브런치 글 기반 광고 운영 실험

> 목적: 메타 + 구글 광고 직접 운영 경험 → 이력서 반영  
> 브런치 글: https://brunch.co.kr/@aha-forest/12

---

## 전체 구조

```
메타 광고 (UTM: utm_source=meta)
구글 광고 (UTM: utm_source=google)
        ↓
HTML 랜딩 페이지 (Vercel 배포) ← index.html
  - GA4 설치 → 세션/UTM 추적
  - Meta Pixel 설치 → 광고 최적화
  - 글 티저 + CTA 버튼
        ↓
브런치 글 전문
```

---

## 진행 순서

### Step 1. GA4 계정 생성
- analytics.google.com → 새 속성 생성
- 측정 ID 발급 (`G-XXXXXXXXXX`)
- `index.html` 안의 `G-XXXXXXXXXX` 두 곳 교체

### Step 2. Meta Business Manager 설정
- business.facebook.com → 계정 생성
- 픽셀 생성 → Pixel ID 발급
- 인스타그램 계정 연결
- `index.html` 안의 `YOUR_PIXEL_ID` 두 곳 교체

### Step 3. Google Ads 계정 생성
- ads.google.com → 계정 생성
- 신규 계정 쿠폰 확인 (종종 15만원 제공)

### Step 4. Vercel 배포
- vercel.com → Add New Project
- `index.html` 파일 업로드
- 배포 URL 확보 (예: `vibe-landing.vercel.app`)
- ID 교체 후에는 파일 수정 → 재업로드 (30초 반영)

### Step 5. 메타 광고 캠페인 세팅

| 항목 | 설정값 |
|------|--------|
| 캠페인 목표 | 트래픽 |
| 예산 | 일 5,000원 × 7일 |
| 타겟 | 25~40세, 관심사: 마케팅·AI·스타트업·업무효율화 |
| 게재 위치 | 인스타그램 피드 + 릴스 |
| 광고 소재 | 카드뉴스형 이미지 1장 |
| 랜딩 URL (기본) | `https://vibe-landing-jinseonis-projects.vercel.app?utm_source=meta&utm_medium=paid_social&utm_campaign=vibe-coding` |
| 랜딩 URL (A/B - FOMO) | `https://vibe-landing-jinseonis-projects.vercel.app?utm_source=meta&utm_medium=paid_social&utm_campaign=vibe-coding&utm_content=fomo` |

## 검증 방법

1. **GA4**: 배포 후 직접 URL 접속 → GA4 실시간 보고서에서 세션 확인
2. **UTM 추적**: GA4 → 획득 → 트래픽 획득 → source/medium 분리 확인
3. **Meta Pixel**: 크롬 확장 "Meta Pixel Helper"로 PageView 이벤트 발화 확인
4. **광고 집행 후**: 메타 광고 관리자 + 구글 애즈 대시보드에서 CTR, CPC, 도달 확인
