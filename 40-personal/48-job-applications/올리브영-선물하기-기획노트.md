# 올리브영 선물하기 기획 노트

> 작성일: 2026-03-31
> 목적: Q3 마케팅 아이디어 기획 과정 기록

---

## 전제: 현재 소비자의 선물 행동

- 대부분의 선물은 **올리브영 금액권** 형태
- 이유: 받는 사람이 필요할 때 원하는 걸 직접 고르게 하기 위해
- 생일 당일이 아니라 필요한 시점에 사용
- 현대의 선물 = 감정적 행위가 아닌 **사회생활 의례**

---

## 기각된 아이디어와 이유

| 아이디어 | 기각 이유 |
|---------|---------|
| 감정 전달 (나 너 생각했어) 강조 | 현대 선물은 감정보다 효율이 중심. 소비자가 체감할 필요 없음 |
| D-7 생일 푸시 알림 | 올리브영은 소셜 그래프(지인 생일 데이터)가 없음. 실행 불가 |
| 위시리스트 공유 기능 | 카카오 선물하기에서 이미 진행 중. 차별점 없음 |
| 오늘드림 연계 당일 선물 | 올리브영 선물하기에 이미 배송/매장 픽업 옵션 존재 |

---

## 현재까지 도달한 핵심 인사이트

### 놓친 본질 (가설)
- 금액권 사용 행동은 이미 올리브영 ecosystem 안에서 일어남
- 그러나 **"선물하기" 퍼널을 통하지 않음** — 두 경험이 분리되어 있음
- 올리브영이 선물 트리거 시점에 없음 (카카오가 소셜 그래프로 선점)

### 탐색 중인 방향
- 금액권 행동 자체를 선물하기 퍼널 안으로 끌어들이기
  - 발신자: 금액만 설정 → 선물하기로 전송
  - 수신자: 앱에 들어와 직접 선택
  - 수신자가 반드시 앱을 열어야 하는 구조
- **아직 결론 미도달 — 추가 논의 필요**
