## 섹션 1. 매체별 초기 0→1 운영 신규 계정 세팅

**배경**
광고 기획 경험은 있었으나 직접 세팅·운영 경험이 없어 소예산으로 SA/DA 전 과정을 직접 구현했습니다.
카카오 브런치 글 유입을 목표로 메타(DA)와 구글(SA)을 동시 운영하며 채널별 특성을 검증했습니다.

### 1. Claude Code를 활용한 HTML 랜딩페이지 제작 및 Vercel 배포

- 최종 전환 이벤트가 카카오 브런치 플랫폼으로 연결되어, 정확한 데이터 분석 위한 랜딩페이지 구축 필요성 제기

**계측 설계 (집행 전 선행)**

수동 gtag.js 방식 대신 GTM 컨테이너로 태그를 일원화해 이후 수정·확장을 코드 배포 없이 가능하도록 설계했습니다.

```
GTM (GTM-N8NS54Z2)
├── GA4 설정 태그 (G-VNB04L30DF) — 전 페이지 세션 수집
├── GA4 이벤트 태그 — cta_click (브런치 CTA 클릭)
└── Meta Pixel (1651900122739158) — 광고 최적화·리타겟팅
```

UTM 4단계 구조로 채널·소재 변수를 사전에 분리:
`utm_source / utm_medium / utm_campaign / utm_content`

### 2. 수동 gtag.js 방식에서 GTM 태그 변경 설계

- 랜딩페이지에서의 이탈률 확인을 위한 GTM 이벤트 태그 - `cta_click` (브런치 CTA 클릭) 추가

### 3. 메타_DA (A/B) · 구글_운영 캠페인 효율 (PMAX/검색)

- 소예산(매체 별 4만원) 진행 시 머신러닝 최적화 기간 중 구글 16건의 전환(클릭) 발생했으나, GA4 유입 및 구글 이벤트 매개변수 확인 결과 **정치 뉴스 지면 위주 노출 + 해외 유입 과다** 확인 — 타겟 지면 불일치로 학습 모수 신뢰 불가 판단, 조기 중단

## 섹션 2. A/B 테스트 분석

*(Meta 기준\_클릭 : 내+외부페이지)*

**타겟 USP 설정**

| 항목 | 내용 |
|------|------|
| 타겟 고객 | 반복 업무에 지친 마케터, AI 도구 관심 있지만 진입 어려운 사람 |
| 핵심 USP | "코딩 몰라도 AI로 마케팅 자동화 가능하다" |
| 콘텐츠 | 브런치 글 — 실무 마케터가 직접 쓰는 AI 자동화 방법 |

### 소재 A — USP: 사람·심리자극

**[결과]** 노출과 도달 모두 소재 B보다 우위이나, 비용 대비 전환 미흡

| 지표 | 수치 |
|------|------|
| 클릭 | 66회 (CTR 2.16%) |
| 노출 | 3,049회 |
| 도달 | 2,523회 |
| CPLPV | 331원 |

- 이미지: 기존 콘텐츠 연동

### 소재 B — USP: CTA·정보제공 ★우위

**[결과]** 비용 대비 전환 우위

| 지표 | 수치 |
|------|------|
| 클릭 | 74회 (CTR 2.95%) |
| 노출 | 2,512회 |
| 도달 | 2,011회 |
| CPLPV | 308원 |

- 이미지: AI 및 피그마 활용 제작

**A/B 핵심 인사이트**

> 메타 자동 보고서는 CPM 기준으로 소재 A 우위 판정. 그러나 캠페인 목표 기준 지표(CTR, CPLPV)로 재분석 시 소재 B가 효율 우위 — 플랫폼 자동 판단과 목표 기반 판단의 차이를 직접 검증.
> CTA-정보제공 소재가 CTR 2.95% vs 2.16%, CPLPV 308원 vs 331원으로 효율 우위.

## 섹션 3. GA4 운영 결과 퍼널 분석

*(GA4 기준)*

```
광고 클릭      144명   랜딩페이지 방문
     ↓
랜딩 진입      92명    브런치 방문 (cta_click)
               52명    랜딩 이탈자
     ↓
결과           클릭 후 실제 방문율 63.9%
```

**분석 구조**

```
[광고 클릭] → [랜딩 진입] → [CTA 클릭] → (브런치 이후 계측 한계)
                ↑ 방문율       ↑ cta_click율
             광고관리자        GA4 이벤트
```

**채널별 결과 (메타)**

| 지표 | 수치 |
|------|------|
| 집행 기간 | 7일 완료 |
| 소진 예산 | 35,000원 |
| 랜딩 페이지 조회 | 128명 (소재A 63 + 소재B 65) |
| CPLPV 평균 | 약 273원 |

**학습 및 인사이트**

1. **구글 SA**: 소예산 환경에서 러닝 기간 비용 과다 + 정치 뉴스 지면 노출 확인. 지역 타겟 한국 제한 및 수동 CPC 입찰 설정이 선행되어야 함
2. **PMax vs SA**: 타겟 제어가 필요한 니치 오디언스 캠페인에서 PMax는 비효율. 지면 매개변수로 실제 노출 지면 확인 필수
3. **메타 vs GA4 수치**: 플랫폼별 측정 기준 차이로 수치 불일치는 정상. 각 지표 출처를 구분하여 교차 해석
4. **계측 한계**: 브런치 도달 이후 구간은 외부 플랫폼 특성상 GA4 추적 불가. 자체 랜딩→전환 루프 구조 필요
