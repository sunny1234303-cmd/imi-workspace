# 05. 카피 원고 모음

---

## ✅ 확정 메인 카피

```
마케팅을 운영하지 않습니다.
마케팅이 작동하는 구조를 만듭니다.
```

**선택 이유**
- "운영 vs 구조" 대비가 선명 → 한 번 읽으면 잊히지 않음
- 일하는 방식(사고방식)을 보여줌 → 입사 후 어떻게 일할지 그대로 전달됨
- 과장 없이 증명 가능한 문장 → 뒤에 오는 케이스 전체가 이 한 줄의 근거

---

## ✅ 첫 장표 (커버 슬라이드) — PPT/PDF

### 레이아웃 구조

```
┌──────────────────────────────────────────────────────┐
│                                                      │
│  [사진]                                              │
│  세로로 긴 직사각형,        마케팅을 운영하지 않습니다.  │
│  왼쪽 40% 차지             마케팅이 작동하는            │
│  (전문적·자신감 있는        구조를 만듭니다.            │
│   표정, 단색 배경)                                    │
│                            ───────────────           │
│                            진서니                    │
│                            Brand Marketer            │
│                                                      │
│  ────────────────────────────────────────────────    │
│                                                      │
│      KPI 153%         99% 단축        ROAS +152%     │
│    예산 0원 캠페인    반복업무 자동화   IMC 통합운영    │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 레이아웃 원칙
- **좌: 사진** (40%) — 인물 중심, 세로형, 단색 배경
- **우상: 메인 카피** — 가장 크고 굵게 (28~32pt Bold)
- **우중: 이름 + 직함** — 카피보다 작게 (14pt)
- **하단 전체: 숫자 3개** — 각각 독립된 블록, 숫자 크게 (36pt+) / 설명 작게

### 컬러 방향
| 요소 | 컬러 |
|------|------|
| 배경 | 화이트 (#FFFFFF) |
| 메인 카피 | 다크 네이비 (#1E2D40) — 묵직하고 신뢰감 |
| 숫자 강조 | 포인트 컬러 1개 선택 (딥블루 또는 차콜) |
| 이름·직함 | 미디엄 그레이 (#666666) |

### 타이포그래피
| 요소 | 폰트 | 사이즈 |
|------|------|--------|
| 메인 카피 | Pretendard ExtraBold | 28~32pt |
| 숫자 | Pretendard Bold | 36~40pt |
| 숫자 설명 | Pretendard Regular | 11~12pt |
| 이름 | Pretendard SemiBold | 16pt |
| 직함 | Pretendard Light | 12pt |

### 핵심 원칙
> 채용 담당자가 3초 안에 읽을 수 있어야 한다.
> 사진 → 카피 → 숫자 순서로 시선이 흐르게 설계한다.
> 텍스트는 이것 이상 넣지 않는다.

## 자기소개 문장 — 용도별

### 커버 슬라이드 바로 다음 장 (2페이지)

```
스타트업 창업부터 글로벌 브랜드 인하우스 마케터까지,
어떤 환경에서도 같은 방식으로 일해왔습니다.

채널이 흔들리면 채널 구조를 바꾸고,
반복 업무가 생기면 자동화 시스템을 만들었습니다.
AI 도입이 막히면 보안 없이 쓰는 방법을 만들었습니다.

추가 예산 없이 캠페인 KPI 153%를 달성했고,
대기업 AI 전환 담당자가 먼저 자문을 요청해왔습니다.
```

### 링크드인 헤드라인 (한 줄)

```
마케팅이 작동하는 구조를 만드는 마케터 | 브랜드 전략 × AI 자동화
```

### 면접 30초 자기소개

```
저는 마케팅을 설계하는 사람입니다.

채널이 막히면 채널 구조를 바꾸고,
도구가 없으면 직접 만들어왔습니다.

예산 없이 KPI 153%를 달성했고,
대기업 AI 전환 담당자가 먼저 찾아왔습니다.

그게 제가 일하는 방식입니다.
```

### 이력서 한 줄 요약

```
브랜드 전략 설계부터 AI 자동화 시스템 구축까지 단독 수행한 마케터
```
