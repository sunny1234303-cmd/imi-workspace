### 추천: 드로잉 입문 (임진석)

**이유:**
1. **타겟 범위가 가장 넓다** — 드로잉을 배우다 막힌 사람은 어디에나 있음. 성별/연령 불문
2. **페인포인트가 제목에 이미 박혀 있다** — "정체기 탈출"은 그 자체로 카피. 타겟이 즉각 반응
3. **차별점이 명확하다** — "감이 아닌 공식" = 기존 감성 드로잉 클래스들과 포지션이 다름
4. **Meta 광고 소재 쓰기 가장 쉽다** — 정체기 공감 → 공식 전달 → CTA로 이어지는 구조 자연스러움
5. **과제 목적에 부합** — 콘텐츠 마케터 역량(타겟 설정, 카피, 광고 전략)을 가장 풍부하게 보여줄 수 있는 소재

---

### ✅ 최종 확정 버전 (V1+V2 혼합)

#### 1a. 메인 타겟
```
머릿속의 그림을 그대로 그리고 싶은 드로잉 학습자
```

#### 1b. 페인포인트
```
열심히 그렸는데 입체감이 부족하고 완성도가 낮다 
```

#### 1c. 셀링 방향

원기둥, 육면체, 구 같은 기초 도형부터 금속, 천의 질감 표현까지 — "이렇게 따라 그려라"가 아닌 **형태가 입체가 되는 원리와 구도를 보는 접근법**을 알려준다. 핵심 원리를 이해하면 어떤 대상이든 스스로 응용할 수 있다는 점에서, 단순 스킬 전달이 아닌 **드로잉을 보는 눈을 키우는 강의**로 포지셔닝한다.

#### 2. 셀링포인트 한 줄
```
도형부터 질감까지, 원리로 완성하는 드로잉 (21자)
```

#### 광고 카피 (별도 활용)
```
생각대로 안 그려진다면, 원리가 없는 겁니다
```

### 3. 동일 카테고리 대비 수강 이유

**경쟁 강의 포지션**: 인물 그리기(스타일·테크닉), 완전 기초 도형(따라 그리기), 색감/오일파스텔(재료·감성)

> 콜로소 드로잉-손글씨 카테고리의 강의는 인물 드로잉, 입체감 및 질감, 채색처럼 **"멈춰 있는 그림"** 을 가르칩니다. 해당 강의는 도형이 입체가 되는 방법, 형태에 역동적인 움직임을 부여하는 매커니즘을 기반으로 어떤 대상이든 완성도 있고 살아있는 그림으로 만드는 접근 원리를 강의합니다. 드로잉의 본질을 기초부터 이해하는 강의로 포지셔닝합니다.

## Meta 광고 작성 방향

### Text (피드 본문)
```
생각대로 안 그려진다면, 기본기가 없는 겁니다.
도형이 입체가 되는 원리, 형태에 움직임을 부여하는 구조 —
이걸 모르면 아무리 그려도 제자리입니다.
드로잉의 본질부터 배워보세요.
```

### Title
```
머릿속 그림을 완성하는 드로잉 원리 강의
```

- 강의 화면보다 결과물 이미지 우선

## 카테고리 전략 방향

> 카테고리: 드로잉/일러스트 (취미 창작)

### 광고 전략

**유효한 전략: 페인포인트 기반 피드 광고 + 강사 팔로워 연계 퍼널**

**카테고리 분석:**
콜로소 드로잉 카테고리 서브 분류(캐릭터 / 웹툰·만화 / 인체·포즈 드로잉 / 배경 / 컨셉아트·디자인 / 손그림 / 아이패드 드로잉)를 보면, 구매자는 단순 취미보다 목적이 있는 사람들입니다. 웹툰 작가, 캐릭터 디자이너, 게임 아트 지망생 등 실력 향상에 진지하게 투자하는 층입니다.

**이유:**
1. **목적형 구매자** — 충동보다 필요에 의해 삽니다. "내가 왜 이 강의가 필요한지"를 납득시키는 페인포인트 카피가 클릭률에 직결됩니다.
2. **비포/애프터 소재** — 드로잉 카테고리는 결과물이 시각적으로 명확합니다. 원리를 모를 때 vs 알고 난 후 그림 비교가 Meta에서 CTR 높습니다.
3. **강사 팔로워 → 강의 유입** — 드로잉 카테고리는 강사 신뢰도가 구매 전환의 핵심입니다. 강사 SNS 팔로워를 콜로소 랜딩으로 연결하는 퍼널이 가장 높은 전환율을 만듭니다.
4. **리타겟팅** — 강의 상세페이지 방문자 대상 할인/얼리버드 메시지로 이탈 방어합니다.

### 매출 전략

**유효한 전략: 뾰족한 타겟 포지셔닝 기반 전환 최적화 + AI 강의 매칭 서비스**

**이유:**

1. **난이도·목적 매칭이 전환율 결정** — 드로잉 카테고리 구매자는 "이게 나한테 맞는 강의인가"를 먼저 판단합니다. 상세페이지에서 타겟(머릿속 그림을 그리고 싶은 학습자)과 페인포인트(입체감 부족, 어설픈 완성도)를 명확히 명시해 "이건 내 얘기다"라는 즉각 반응을 유도합니다.

2. **강의 포지셔닝 차별화** — 카테고리 내 경쟁 강의(인물/채색/손그림)와 겹치지 않는 "드로잉 본질·원리 이해" 포지션을 선점합니다. 명확히 다른 포지션이 이탈 없이 클릭을 전환으로 연결합니다.

3. **AI 강의 매칭 전환 전략** — 드로잉 카테고리 구매 이탈의 핵심 원인은 "이게 내 수준에 맞는 강의인가"라는 확신 부재입니다. 콜로소 드로잉 카테고리 진입 시 AI 챗봇이 현재 실력, 목표(캐릭터/웹툰/취미 등), 가장 큰 고민 3가지를 묻고 "지금 당신에게 맞는 강의는 이것입니다"를 즉시 추천합니다. 확신을 주는 순간 이탈이 전환으로 바뀌고, 매칭 결과 SNS 공유 유도로 오가닉 유입까지 확보합니다.

4. **수강 후기 소셜 증거** — 실력 향상이 시각적으로 보이는 카테고리인 만큼, 수강 전후 비교 후기가 신규 구매자 전환에 직접 영향을 줍니다.
