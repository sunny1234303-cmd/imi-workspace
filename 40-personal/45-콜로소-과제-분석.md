### 추천: 드로잉 입문 (임진석)

**이유:**
1. **타겟 범위가 가장 넓다** — 드로잉을 배우다 막힌 사람은 어디에나 있음. 성별/연령 불문
2. **페인포인트가 제목에 이미 박혀 있다** — "정체기 탈출"은 그 자체로 카피. 타겟이 즉각 반응
3. **차별점이 명확하다** — "감이 아닌 공식" = 기존 감성 드로잉 클래스들과 포지션이 다름
4. **Meta 광고 소재 쓰기 가장 쉽다** — 정체기 공감 → 공식 전달 → CTA로 이어지는 구조 자연스러움
5. **과제 목적에 부합** — 콘텐츠 마케터 역량(타겟 설정, 카피, 광고 전략)을 가장 풍부하게 보여줄 수 있는 소재

## 항목별 작성 프레임워크

> 강의 상세페이지 확인 후 구체화 필요. 아래는 방향성 초안.

### 1a. 메인 타겟

**방향**: 드로잉을 배우고 싶거나 배우고 있지만 실력이 늘지 않는다고 느끼는 사람

```
초안: 실력이 늘지 않아 좌절한 취미 드로잉 학습자
→ 공백 제외 약 22자 (적정)
```

### 1b. 타겟 페인포인트

**방향**: 오래 그려도 늘지 않는다는 좌절감 / 어디서부터 잘못됐는지 모른다

```
초안: 열심히 그려도 실력이 그대로인 드로잉 정체기
→ 공백 제외 약 23자 (적정)
```

### 1c. 셀링 방향

**방향**: "감(感)"이 아니라 "공식"으로 배울 수 있다는 논리적 접근으로 신뢰감 획득
- 기존 드로잉 강의들이 "따라 그리기" 중심인 것과 달리, 원리 기반 학습임을 강조
- 결과물(그림 퀄리티 향상)보다 과정(왜 못 그리는지 파악 + 교정)에 집중한 포지셔닝
- 타겟이 "이번엔 다르다"는 확신을 갖도록 유도

### 2. 메인 셀링포인트 한 줄

```
초안: 감으로는 못 뚫는 정체기, 공식으로 뚫는다
→ 공백 제외 약 20자 (적정)
```

### 3. 동일 카테고리 대비 수강 이유

**방향**: 콜로소 드로잉 카테고리 내 다른 강의들은 스타일/테크닉 중심인데, 이 강의는 "왜 늘지 않는지"를 설명하는 구조 진단형 커리큘럼이 핵심 차별점

> ⚠️ 강의 상세페이지에서 커리큘럼 구성, 강사 포지셔닝, 후기 확인 후 구체화 필요

## Meta 광고 작성 방향

### Text (피드 본문)
```
드로잉, 몇 년째 하고 계세요?

그런데 아직도 "느낌"으로 그리고 있다면
실력이 안 느는 게 당연합니다.

임진석 작가가 알려주는 건
테크닉이 아니라 공식 —
내 그림이 어디서 막혔는지 정확히 짚어냅니다.

정체기, 이번엔 진짜 뚫어보세요.
```

### Title
```
초안: "감 말고 공식으로 — 정체기 탈출 드로잉"
```

### Image 선택 기준
- 비포/애프터 비교가 보이는 이미지 (정체기 → 향상 시각화)
- 강사의 설명 장면 (공식 전달 느낌)
- 강의 화면보다 결과물 이미지 우선

### 사용 툴
- **Canva** (PPT 대용, 실사용 가능한 툴)
- PPT 선택 시: 가장 무난하고 실사용 입증 가능

## 카테고리 전략 방향

> 카테고리: 드로잉/일러스트 (취미 창작)

### 광고 전략

**유효한 전략: 페인포인트 기반 영상 광고 (릴스/숏폼) + 리타겟팅**

**이유:**
- 드로잉 학습자는 유튜브/인스타에서 다른 작가 작품을 보며 욕구가 발생 → SNS 광고 노출 시 즉각 반응
- "정체기"라는 공통 경험은 댓글/공감 유도가 쉬워 오가닉 확산 가능
- 짧은 비포/애프터 영상이 Meta에서 CTR 높음
- 리타겟팅: 강의 상세페이지 방문자 대상 할인 쿠폰/얼리버드 → 전환 유도

### 매출 전략

**유효한 전략: 시즌 기반 번들 + 강사 SNS 팔로워 연계 퍼널**

**이유:**
- 취미 강의는 충동 구매 비율이 높음 → 할인/기간 한정 메시지가 구매 유도에 효과적
- 강사 SNS 팔로워 → 콜로소 랜딩 유입 퍼널이 창작 분야에서 가장 높은 전환율
- 번들 (드로잉 입문 + 심화 패키지): LTV 높이기 + 이탈 방어
- 수강 완료 후기 콘텐츠 제작 → 신규 유입 소셜 증거로 활용
