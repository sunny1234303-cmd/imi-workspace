## 상세페이지 초안

# 마케터를 위한 Claude Code로 업무 자동화 워크플로우 만들기

## 한눈에 보기

- **카테고리**: 업무&비즈니스
- **난이도**: 입문 🐥
- **기간**: 4주
- **도구**: Claude Code + LLM (ChatGPT, Gemini, Claude)
- **결과물**: 나만의 마케팅 업무 자동화 워크플로우 1개

## 이런 분께 추천해요

- 자동화 아이디어는 없어도 OK, **본인의 업무를 잘 알고 있는** 마케팅 실무자
- Claude Code는 깔았는데 뭘 자동화해야 할지 모르겠는 분
- 코딩은 못하지만 자연어로 업무 도구를 만들어보고 싶은 마케터
- 혼자 하면 막막한데 같이 고민하면서 만들어보고 싶은 분

## 이런 문제를 바이브코딩으로 해결해요

- **매주 같은 리포트 수작업** → Claude Code에게 말로 설명해서 자동 생성
- **여러 플랫폼 데이터 취합 노가다** → 자연어 한 줄로 수집/정리 자동화
- **뭘 자동화할 수 있는지 모름** → 내 업무 분석부터 기획까지 함께 설계
- **만들고 싶은 건 있는데 어떻게 시작할지 모름** → 기획서 → 프롬프트 → 실행까지 단계별 진행

## 바이브코딩으로 만드는 결과물

4주 후 가져갈 것:
- **나만의 업무 자동화 워크플로우 1개** - 실제 내 마케팅 업무에 바로 적용
- **자동화 기획력** - "뭘 자동화할까?"를 스스로 판단하는 프레임워크
- **Claude Code 실무 활용법** - 마케터 관점에서의 자연어 자동화 경험
- **동료 마케터들의 자동화 사례** - 다양한 업무 영역의 실제 적용기

**주요 내용**

- 자동화 ROI가 높은 업무 우선순위 정하기
- Claude Code 기본 사용법 복습

**실습 과제**: 내 업무 중 자동화할 타겟 1개 선정 + 이유 공유

**결과물**: 나의 자동화 후보 리스트 & 최종 타겟 1개

## 바이브코딩 도구 소개

**Claude Code**는 터미널에서 자연어로 코딩하는 CLI 도구입니다.
코드를 직접 쓰지 않아도 "이런 자동화 만들어줘"라고 말하면 Claude가 만들어줍니다.

마케터에게 Claude Code가 좋은 이유:
- **자연어로 지시** - 코딩 문법 몰라도 됨
- **파일/데이터 처리** - CSV, 엑셀, API 데이터 자동 처리
- **스킬 시스템** - 한 번 만들면 반복 사용 가능
- **API 연동** - 구글 시트, 노션, 슬랙 등 외부 서비스 연결
