## 1. 핵심 포지셔닝

### 메인 카피
```
마케팅을 운영하지 않습니다.
마케팅이 작동하는 구조를 만듭니다.
```

### 한 줄 정의
> **"마케팅 시스템을 설계하는 사람"**

### 포지셔닝 키워드 3개
- **설계**: 채널 구조, 브랜드 USP, 자동화 시스템을 기획부터 만든다
- **자동화**: AI 활용 자동화 (보안 해결 방법론 포함 — 더미데이터 방법론)
- **전환**: 모든 기획의 최종 목적은 구매 전환과 매출

### 타깃 기업별 어필 포인트
| 타깃 | 그들의 니즈 | 내가 줄 수 있는 것 |
|------|------------|-------------------|
| 대기업 | 검증된 실행력 + 시스템 이해 | 수치 성과 + 보안 이슈 없는 AI 도입 방법론 |
| IT/테크 스타트업 | 자율 실행 + 도구 활용 | 채널 설계 + 자동화 구축 + SaaS 개발 경험 |
| 리테일/소비재 | 퍼포먼스 + 전환 최적화 | IMC 통합 운영 + CRM + 상세페이지 리뉴얼 |
| 브랜드/콘텐츠 | 브랜드 감각 + 실행 | 인플루언서 협상 + 콜라보 기획 + 콘텐츠 전략 |

## 2. 핵심 경험 4개 (스토리 각도 포함)

### Case 1. 새학기 캠페인 (2025.02)
**기억에 남길 한 문장**: "돈이 없어서, 브랜드 간 연결고리를 직접 찾았다."

**핵심 수치**
- 전년 동월 대비 신규 구매 고객 **+47.4%**
- 월 매출 **+41.3%** (YoY)
- 추가 예산 **0원**

**스토리 흐름**
1. 상황: 예산 추가 없음 + 키즈 성수기 + 신규 고객 필요
2. 발견: 내셔널지오그래픽 × YBM — 공통점은 '영어교육'
3. 설계: 타깃을 '독자' → '초등 자녀를 둔 학부모'로 재정의 → YBM 초등 영어회화 무료 체험권 혜택 기획 + 구매 금액별 사은품 증정으로 객단가 증대 구조 병행
4. 결과: 신규 고객 YoY +47.4%, 월 매출 YoY +41.3%

**적용 직무**: IMC/브랜드 캠페인 기획, 콜라보 마케팅, 퍼포먼스 마케팅

### Case 2. 과월호 골라담기 캠페인 (2025.06~08)
**기억에 남길 한 문장**: "매출이 떨어지는 게 아니라, 구조가 잘못된 거였다."

**핵심 수치**
- 일 평균 대비 매출 **+203%**
- CRM 발송 후 네이버 검색 유입률 **+143%**
- 재활성화한 휴면 DB: **7,300명**

**스토리 흐름**
1. 상황: 비수기 매출 급락
2. 발견: 네이버 검색광고 단일 채널 의존이 구조적 문제
3. 설계: 2년간 휴면 카카오 채널 DB 7,300명 재활성화 + 네이버 스마트스토어 알림받기 CRM 연동 + 과월호 번들 큐레이션 신상품 기획·런칭 (회사 최초 과월호 온라인 판매 채널 개척)
4. 결과: 일 평균 대비 매출 +203%, 검색 유입 +143%

**적용 직무**: CRM 마케팅, 퍼포먼스 마케팅, 이커머스, 채널 전략

### Case 3. 어린이날 콜라보 & 인플루언서 캠페인 (2025.04~05)
**기억에 남길 한 문장**: "예산이 없으면 협상하면 된다."

**핵심 수치**
- 키즈 인플루언서 직접 컨택·계약·운영: **11명**
- 원고료 **0원** (협찬 기반 계약)
- 사입 단가 **38% 절감** (그란데클립 협상)

**스토리 흐름**
1. 상황: 인스타그램에 상업적 공구 포스팅이 상위 노출 → 브랜드 이미지 오염
2. 발견: 그란데클립 — 젊은 부모 팬덤 보유 + 내셔널지오그래픽 — 콘텐츠 자산
3. 설계: '공룡' 테마 콜라보 기획 → 사입 단가 38% 협상 → 인플루언서 11명 원고료 없이 릴스 집행 → 잉여 재고 네이버 특선물 프로모션 연결
4. 결과: 연관 키워드 검색 시 실 체험 콘텐츠 11편 상위 노출, 조직 내 최초 인플루언서 채널 구축

**적용 직무**: 인플루언서 마케팅, 콘텐츠 마케팅, 브랜드 마케팅, 콜라보 기획

### Case 4. AI 자동화 & 더미데이터 방법론 (2024.07~2025.11)
**기억에 남길 한 문장**: "데이터를 주지 않고도 AI를 쓰는 방법이 있다."

**핵심 수치**
| 업무 | 기존 | 개선 후 | 단축률 |
|------|------|---------|--------|
| 월 정산·리포팅 | 4시간 | 클릭 1번 | 99% |
| 이미지 에셋 제작 | 2일 (유관부서 의뢰) | 10분 이내 | 99%+ |
| 퍼포먼스·시즈널 분석 | 3시간 | 1분 이내 | 97% |

**스토리 흐름**
1. 상황: AI 도입하고 싶지만 내부 데이터 보안 이슈
2. 발견: 더미데이터로 AI에 구조를 학습시키면 실제 데이터 없이 자동화 가능
3. 설계: Excel VBA + LLM AI (더미데이터), 구글 시트 App Script RPA 직접 구축
4. 결과: 작업시간 99% 단축, 대기업 가구 브랜드 AX 자문 인바운드 수주

**외부 검증**: 대기업 가구 브랜드 AX 자문 (2025.12~2026.02) — 마케팅팀 AX 구조화 프레임워크 설계·제안 → 사내 AX 검토 자료로 활용

**적용 직무**: 마케팅 자동화, 퍼포먼스 마케팅, 스타트업/IT, AI 활용

## 4. 면접 예상 Q&A

### Q. "2년차인데 이렇게 굵직한 업무를 했다는 게 맞나요?"
> "기획·제안·실행을 단독으로 수행했습니다. 연차가 아닌 임팩트 기준으로 일해왔습니다.
> 예산 제약, 채널 구조 문제, 자동화 필요성을 직접 발견하고 해결해왔습니다."

### Q. "사원 업무도 괜찮겠어요?"
> "사원 업무도 어떻게 시스템화하느냐에 따라 팀 전체 효율을 바꿀 수 있습니다.
> 저는 반복 업무를 99% 단축하는 자동화를 혼자 만들어봤습니다.
> 주어진 일을 잘 하면서, 구조적으로 더 잘할 수 있는 방법을 함께 찾겠습니다."

### Q. "AI를 어떻게 활용하나요?"
> "단순히 글 쓰는 용도가 아닙니다. 보안 이슈 없이 자동화 시스템을 구축하는 방법론을 직접 만들어 썼습니다.
> 더미데이터로 AI를 학습시켜 VBA 자동화, App Script RPA를 구현했습니다.
> AI를 아직 못 쓰는 회사에서 도입 가능한 방법이 있습니다."

### Q. "창업 경험이 마케터로서 어떤 도움이 됐나요?"
> "사업을 설계할 때 고객 관점에서 문제를 정의하고, MVP로 빠르게 검증하는 방식을 배웠습니다.
> 지금도 캠페인을 기획할 때 '이게 정말 고객 문제를 해결하는가'를 먼저 묻습니다."
