기본 활용법

01

투두 추가

상단 입력창에 할 일을 입력하고 Enter를 누릅니다. 우선순위(높음·중간·낮음)를 함께 설정할 수 있습니다. Esc를 누르면 입력창이 닫힙니다.

Enter 추가 · Esc 접기

02

완료 체크 & 히스토리

항목 왼쪽의 원을 클릭하면 완료 처리됩니다. 완료된 항목은 날짜별 히스토리에 자동 기록되며, 히스토리 패널에서 언제든 돌아볼 수 있습니다.

완료 내역 날짜별 기록

03

그룹 관리

URL 그룹으로 분류해 관리할 수 있습니다.

URL 기반 그룹 관리

04

Google Calendar 연동

체크리스트의 투두를 Google 할 일(Tasks)로 자동 동기화합니다. Google Calendar에서 할 일이 등록되며, 앱 안에서 오늘의 일정 확인·이벤트 추가도 가능합니다.

투두 → Google Tasks 동기화

05

메모 & Apple Notes

화면 하단 메모 영역에 오늘의 생각을 입력합니다. 저장 즉시 Apple Notes 체크리스트 폴더에 동기화됩니다. 투두 목록도 체크박스 형태로 함께 저장됩니다.

Apple Notes 연동 시 자동 저장

06

데이터 저장 방식

모든 데이터는 내 컴퓨터·브라우저에만 저장됩니다. 외부 서버로 전송되지 않습니다.

  • 브라우저 localStorage에 자동 저장
    checklist-todos-v1
    checklist-memo-날짜-v1
  • 서버·앱 앱 폴더 내 JSON 파일로 백업
    checklist-todos.json — 투두 목록
    checklist-history.json — 완료 히스토리
    checklist-log.json — 활동 로그
로컬 저장 · 외부 전송 없음

Windows 10/11 · 교차 기기 동기화

데스크톱 앱 설치 가이드

01 앱 다운로드

아래 버튼으로 앱을 다운로드합니다. Mac 칩에 맞는 버전을 선택하세요.

↓ Apple Silicon (M1/M2/M3) ↓ Intel Mac

내 Mac 칩 확인: 화면 좌상단 애플 아이콘(🍎) → 이 Mac에 관하여 → "칩" 또는 "프로세서" 항목 확인

02 설치

다운로드된 .dmg 파일을 더블클릭해 열고, Checkmate 아이콘을 응용 프로그램 폴더로 드래그합니다.

"개발자를 확인할 수 없음" 오류가 뜨면:
시스템 설정 → 개인 정보 보호 및 보안 → 하단 "그래도 열기" 또는 "확인 없이 열기" 클릭

03 실행

Launchpad 또는 응용 프로그램 폴더에서 Checkmate를 실행합니다. 화면 우측 하단에 TODO 버튼이 나타납니다.

버튼은 항상 화면 위에 표시됩니다. 드래그해서 위치를 바꿀 수 있고, 클릭하면 체크리스트가 열립니다.

01 클론 및 의존성 설치
git clone https://github.com/YOUR_USERNAME/checkmate.git cd checkmate npm install
02 개발 실행
npm start

화면 우상단에 TODO 버튼이 나타납니다.

03 .dmg 빌드 (배포용)
npm run build

dist/ 폴더에 Checkmate.dmg가 생성됩니다. GitHub Releases에 업로드하거나 직접 배포할 수 있습니다.

Windows 앱

01 설치 파일 다운로드

Windows 10/11 (64-bit) 전용 설치 파일입니다. 아래 버튼을 클릭해 다운로드하세요.

Checkmate 다운로드 (.exe) →
02 설치 및 실행

다운로드한 Checkmate Setup 1.0.0.exe를 실행하면 설치 마법사가 시작됩니다. 설치 후 시작 메뉴 또는 바탕화면에서 Checkmate를 실행하세요.

Windows Defender가 경고를 표시할 경우 "추가 정보" → "실행"을 클릭하세요. (코드 서명 인증서 미포함)

03 Windows 제한 사항

macOS와 동일한 기능을 제공하지만, 아래 기능은 Windows에서 지원되지 않습니다.

Apple Notes 동기화 — Windows 미지원. Apple Notes는 macOS 전용 앱으로 Windows에서는 연동이 불가합니다. 투두 관리 · Google Calendar 연동 · 집중 타이머는 정상 지원됩니다.

초기 설정

01 이름 설정

이름을 설정하면 앱 상단에 맞춤 인사말이 표시됩니다.

예: 서니님 간밧테~

02 Google Calendar 연동 선택

앱 안에서 오늘의 일정을 확인하고 이벤트를 추가·삭제할 수 있습니다.

✓ Checkmate 앱 실행 중에 사용 가능합니다.

  • 1. 앱을 실행한 후 우측 상단의 연동 관리 아이콘을 클릭합니다.
  • 2. 연동 패널에서 Google Calendar → 연동하기 버튼을 클릭합니다.
  • 3. 브라우저에서 Google 계정 로그인 창이 열립니다. 사용할 계정으로 로그인합니다.
  • 4. Google이 "캘린더 보기 및 수정" 권한을 요청합니다. 허용을 클릭합니다.
  • 5. 앱으로 돌아오면 오늘의 일정이 자동으로 표시됩니다.

Google Calendar 연동은 Checkmate 앱이 실행 중일 때 동작합니다. 로그인 정보는 내 컴퓨터에만 저장되며 외부로 전송되지 않습니다.

03 Apple Notes 연동 선택 · macOS 전용

메모를 입력하면 Apple Notes의 체크리스트 폴더에 자동 저장됩니다. 투두 항목은 체크박스 형태로 함께 동기화됩니다.

✓ Checkmate 앱을 사용하면 자동으로 연동됩니다. (macOS 전용)

  • 1.Checkmate 앱을 실행합니다.
  • 2.앱 우측 하단의 메모 영역에 내용을 입력합니다.
  • 3.입력 후 잠시 기다리면 Apple Notes가 자동으로 실행되며 체크리스트 폴더에 저장됩니다.
  • 4.처음 실행 시 macOS가 "Notes에 접근 허용" 팝업을 표시합니다. 허용을 클릭하세요.

Apple Notes 연동은 macOS 전용 기능입니다. Windows·Linux에서는 지원되지 않습니다.

주요 기능

01

스마트 투두

우선순위·그룹별로 할 일을 체계적으로 관리합니다. 완료된 작업은 날짜별 히스토리로 자동 기록됩니다.

02

Apple Notes 연동

메모를 입력하면 Apple Notes의 체크리스트 폴더에 자동 저장됩니다. 투두 항목은 체크박스 형태로 동기화됩니다.

03

Google Calendar 연동

오늘의 일정과 투두를 한 화면에서 확인합니다. 캘린더 이벤트 추가·삭제도 앱 안에서 가능합니다.

04

스마트폰에서도 확인

투두와 메모가 Google Calendar·Apple Notes로 자동 동기화됩니다. 외출 중에도 스마트폰으로 오늘의 할 일을 바로 확인하세요.
